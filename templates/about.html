CTAT Tutor Generator

CTAT Tutor Generator is built for Cognitive Tutor Authoring Tools (CTAT) by CMU, which builds cognitive tutors in order to improve student performance. This generator is built to help reduce the workload when building tutors for teaching basic flow of code in programing.

In traditional ways, building a programing cognitive tutor requires creator to manually go through the code step by step as a kind of Example-tracing tutor, which is extremely time consuming especially when a large number of tutors are required. By using is generator, You can either upload your Python code file or edit the Python code online, and a Cognitive Tutor will be generated for downloading or previewing.

Currently, the Generator supports only Python programing language, and the basic primitive types of Python.
